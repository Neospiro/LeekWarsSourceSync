# LeekWarsSourceSync
Noeud de synchronisation entre le FS et LeekWars

##Installer
* Installez nodeJS
* Allez dans le dossier de deploy et tapez `npm install request`
* creez un dossier `srcs` (dossier dans lequel vos sources seront déposées)
* `node LWSS.js`
